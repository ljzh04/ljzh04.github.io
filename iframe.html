Resources Used

YouTube


As a student I’ve been relying on YouTube for most stuff involving tutorials and guides. Although the quality greatly varies across channels, with enough effort, it's likely you can come across one that you can follow or understand well. For this website, the video “The ULTIMATE Guide To Typography For Beginners” and “How To Build A Website From Start To Finish” by DesignSpo was my ultimate guide.


Figma, Dribble, Awwwards, MinimalGallery

Following DesignSpo, I created my first typography, color scheme, and layout rules. I used Figma which is a free popular software for all sorts of design stuff. The design phase of the process may actually be the longest time I spent in building the website as, I kept procrastinating finding more design inspirations. Some websites that I used for ✨inspirations✨ were awwwards.com, dribble.io, and minimal.gallery.


W3Schools, Stack Overflow, ChatGPT XD

The programming part was actually really fun. I only needed to follow my design as closely as possible. I already had a little knowledge about web programming, so it probably helped make my process faster, but because I’m so bad, I still had to spend most of my time debugging stuff.


CodePen, Stack Sorted, YouTube - again

After recreating the design as code, all that remained was to add some polish and effects. Particularly useful were codepen.io and stacksorted.com as it provides complete html, css and javascript demo for visual effects and design ideas. While some channels may not provide complete source code / demos, YouTube remains a useful platform for browsing different design styles / templates / effects which I stole used for ✨inspiration✨ (of course!).

Final Thoughts


After spending days building this website, I feel like there are still a lot to improve. However, time is limited so I need to stop. No seriously, I keep adding stuff, I feel like it's actually unnecessary / too much. But anyway, I feel like I learned a lot of stuff on the jQuery and css/sass side of things... through a bit of copy/paste/edit (of course). The most stressful parts are probably the ‘being creative’ in the design and the debugging bits. But, was it worth it? Yeaaahhhh? It’s a much less stress-inducing pasttime than MOBA games at its lowest. I like the programming part, so I guess I’m just a bit bias, but that’s about it thank you for reading!
